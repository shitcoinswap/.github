# Welcome to Shitcoin Swap

Based in Da Nang, Vietnam, Shitcoin Swap provides system integration services and open-source brokerage tools.

Our flagship product is an [open-source brokerage platform](https://github.com/shitcoinswap/platform), capable of natively interacting with blockchain networks and smart contracts, as well as plugging into traditional finance APIs.
